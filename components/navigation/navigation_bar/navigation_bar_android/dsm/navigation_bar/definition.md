**A navigation bar on Android (aka Bottom navigation bar on Material 2)** is a UI element that enables users to access the primary destinations of a mobile application through three to five persistent tabs. The navigation bar is located at the bottom of the screen, with each destination represented by an icon and an optional text label to facilitate quick switching between top-level sections.  
  
For more information, we invite you to consult the official Android:

-   [**Material 3 Navigation bar documentation**](https://m3.material.io/components/navigation-bar/overview)
    
-   [**Material 2 Bottom navigation bar documentation**](https://m2.material.io/components/bottom-navigation)
